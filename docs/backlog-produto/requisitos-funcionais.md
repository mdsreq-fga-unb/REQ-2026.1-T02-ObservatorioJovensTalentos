# Requisitos Funcionais

!!! info "Unidade 2"
    Esta seção será preenchida na **Unidade 2**, após a consolidação do backlog de produto com o cliente.

Os requisitos funcionais descrevem **o que o sistema deve fazer** — as funcionalidades e comportamentos esperados da plataforma Observatório de Talentos.

Eles serão levantados por meio de entrevistas semiestruturadas com o cliente, workshops de priorização e análise de documentos existentes, e documentados no formato de **User Stories** priorizadas pelo modelo MoSCoW.
